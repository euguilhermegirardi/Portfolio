## Getting started

1.  Prerequisites

- npm

      npm install npm@latest -g

2. Installation

- Clone the repo

      git clone https://github.com/euguilhermegirardi/Portfolio.git

3. Install NPM packages

       npm install

4. Run the application

       npm run start
